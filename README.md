# Análise de Estoque

Programa em Node.js para leitura e análise de arquivos de estoque. Permite processar tabelas, gerar resumos e visualizar os dados de forma organizada.


- `Analizar estoque.bat` → Script para rodar todo o processo de forma automática.

## Como Usar

1. Coloque o arquivo .csv na pasta atual
2. renomeie o arquvo para `dados.csv`
3. Abra o arquivo  `Analizar estoque.bat`
4. feche o terminal somente apos finalizar a ultilizacao do sistema.
